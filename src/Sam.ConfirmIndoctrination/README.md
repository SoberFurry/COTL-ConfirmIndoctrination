# Sam ConfirmIndoctrination

BepInEx-плагин для **Cult of the Lamb**, который предотвращает случайный запуск принятия нового
последователя у круга посвящения. Перед открытием экрана имени/формы показывается подтверждение
«Принять этого последователя в культ?».

- GUID: `com.sam.cultofthelamb.confirmindoctrination`
- Версия игры, под которую собрано: Unity 2022.3.62f2, build 22885603
- Зависимость: BepInEx 5.4.x (пак для COTL). COTL_API не требуется.

## Установка
```
<Папка игры>\BepInEx\plugins\SamMods\ConfirmIndoctrination\
```
Конфиг создаётся при первом запуске: `BepInEx\config\com.sam.cultofthelamb.confirmindoctrination.cfg`.

## Поведение
- При нажатии взаимодействия на **нового ожидающего** последователя появляется компактное окно
  подтверждения. По умолчанию выделена **«Отмена»**.
- Кнопка «Принять» недоступна, пока не отпущена кнопка взаимодействия, плюс короткий debounce
  (по умолчанию 200 мс) — одно нажатие не подтверждает окно.
- **Отмена** (Esc / Back / B): последователь остаётся ждать, ничего не меняется.
- **Принять**: оригинальный экран посвящения открывается ровно один раз (через одноразовый bypass,
  привязанный к ID последователя), затем работает штатно.
- **Повторное посвящение** уже принятого культиста (`Interaction_Reindoctrinate`) **не**
  перехватывается — мод цепляется только к `FollowerRecruit.OnInteract/OnSecondaryInteract`.

## Управление вводом
- Мышь: клик по кнопкам (работает и в Steam Remote Play — мышь стримится).
- Клавиатура: ←/→ — переключение фокуса, Enter/Space — выбрать, Esc/Backspace — отмена.
- Геймпад: южная кнопка — выбрать фокус, восточная — отмена, стик/крестовина — фокус.

## Конфигурация (`...cfg`)
| Параметр | По умолчанию | Описание |
|---|---|---|
| `Mode` | `BeforeScreen` | `BeforeScreen` (безопасно, рекомендуется) / `InsideScreen` / `Both` / `Disabled` |
| `DefaultSelection` | `Cancel` | Кнопка по умолчанию |
| `InputDebounceMs` | `200` | Задержка после отпускания кнопки |
| `AllowCancelForTutorial` | `false` | Не вмешиваться в обучающего/первого последователя |
| `AllowCancelForIntegrations` | `false` | Не вмешиваться в Twitch/интеграционных |
| `VerboseLogging` | `false` | Расширенный журнал (события Appendix A) |

## Важно про режимы
- **`BeforeScreen` (по умолчанию)** — гарантированно безопасный механизм, полностью закрывающий
  задачу «случайное нажатие не запускает принятие». Рекомендуется.
- **`InsideScreen` / `Both`** — добавляет экспериментальную кнопку «Отменить посвящение» **внутри**
  штатного экрана. Она не трогает ванильные кнопки (никаких `RemoveAllListeners`) и закрывает меню
  через штатный `Hide`, подавляя финализацию рекрута. Этот путь **не удалось проверить запуском
  игры** в данной среде (нет автоматизации ввода), поэтому он fail-open и выключен по умолчанию.

## Совместимость
- **Namify / моды имени**: listeners кнопки принятия не трогаются; патч до экрана + отдельная кнопка.
- **Trait Control**: генерация черт не затрагивается.
- **Cult of QoL / CotLMiniMods**: разные методы, конфликтов не ожидается.
- При несовпадении сигнатур после обновления игры мод **fail-open**: не блокирует ванильное действие
  и пишет причину в лог.

## Откат / удаление
`uninstall_sammods.ps1` или удалите папку `BepInEx\plugins\SamMods\ConfirmIndoctrination\`.
